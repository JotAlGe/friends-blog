# FRIENDS - Blog (Proyecto para Práctica Profesionalizante III) 

## Instalación de las dependencias :cd:
> composer install

## Crear base de datos :floppy_disk:
```SQL
    CREATE DATABASE friends;
```
## Correr las migraciones :computer:
```
    php artisan migrate
```
